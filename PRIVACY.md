# Privacy Policy — Load with Prefix

**Last updated:** 2026-03-28

## Data Collection

Load with Prefix does **not** collect, store, transmit, or share any personal data. All data used by the extension (the configured URL prefix) is stored in your browser's synced storage via your Google account and is never sent to any third party.

## Permissions

The extension requests the following browser permissions, used exclusively for its core functionality:

- **Storage** — To persist the URL prefix setting across sessions and devices via Chrome sync.
- **ActiveTab** — To read the current tab's URL and navigate it to the prefixed URL.

## Third-Party Services

This extension does not send data to any third party. Navigation occurs entirely within the browser.
